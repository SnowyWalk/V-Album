# User
- uuid : [Primary Key]
- displayName : 보여지는 닉네임 (중복 - 가능)
- userType : enum UserType { Guest, - Registered }
- externalID : 타 플랫폼에서의 uuid - [Nullable]
- createdAt : 레코드가 생성된 시각
- registeredAt : 가입한 시각 [Nullable]
- updatedAt : 마지막 레코드 수정 시각
- deletedAt : 회원탈퇴 시각 [Nullable]

# Group
- uuid : [Primary Key]
- name : 그룹명
- createdAt : 만들어진 시각
- updatedAt : 마지막 레코드 수정 시각
- deletedAt : 그룹 해체 시각 [Nullable]

# Member (그룹 멤버)
- userUUID : User의 uuid
- groupUUID : Group의 uuid
- role : enum MemberRole { Master, Manager, Member }
- alias : 그룹 별 멤버별명
- [Primary Key] : groupUUID + userUUID
- joinedAt : 가입한 시각
- updatedAt : 마지막 레코드 수정 시각
- deletedAt : 그룹 탈퇴 시각 [Nullable]
- [Index] : (userUUID, deletedAt IS NULL), (groupUUID, joinedAt, deletedAt IS NULL)

# Post (그룹에 쓰여지는 글)
- uuid : [Primary Key]
- groupUUID : 글이 쓰여진 그룹의 Group.uuid
- userUUID : 글 작성자 User.uuid
- title : 제목
- body : 글 내용
- createdAt : 글이 작성된 시각
- updatedAt : 마지막 레코드 수정 시각
- deletedAt : 글 삭제 시각 [Nullable]
- [Index] : (groupUUID, createdAt DESC), (groupUUID, userUUID, createdAt DESC, deletedAt IS NULL)

# PostTag
- postUUID : 태그의 주체가 되는 글의 Post.uuid
- targetUserUUID : 태그 대상 유저의 User.uuid
- [Primary Key] : postUUID + targetUserUUID
- createdAt : 태그 발생한 시각
- [Index] : (postUUID, createdAt)
> PostTag는 삭제 시, Hard delete

# PostComment (그룹에 쓰여진 글의 댓글)
- uuid : [Primary Key]
- postUUID : 이 댓글의 부모가 되는 글의 Post.uuid
- userUUID : 댓글 작성자 User.uuid
- body : 댓글 내용
- createdAt : 댓글이 작성된 시각
- deletedAt : 댓글 삭제된 시각 [Nullable]
- [Index] : (postUUID, createdAt DESC, deletedAt IS NULL)

# Place
- uuid : [Primary Key]
- name : 장소명
- createdAt : 장소가 최초 등록된 시각
- updatedAt : 마지막 레코드 수정 시각
- validatedAt : 연동되는 외부 플랫폼에서의 장소명 변경을 확인한 마지막 시각
- validatedVersion : 연동되는 외부 플랫폼에서 확인된 마지막 버전 (numeric)
- deletedAt : 장소 제거된 시각 [Nullable]

# Photo
- uuid : [Primary Key]
- groupUUID : 이 사진을 게시한 글의 소속 그룹 Group.uuid
- postUUID : 이 사진을 게시한 글의 Post.uuid
- sortOrder : 글 내에서의 사진 순서 index
- placeUUID : 이 사진이 찍힌 장소의 Place.uuid [Nullable]
- src : 사진의 storage 내 경로
- previewSrc : 저화질 미리보기 이미지의 storage 내 경로
- width : 사진 너비
- height : 사진 높이
- size : 사진의 파일 크기
- srcHash : 사진의 해쉬 (중복 업로드 체크용)
- createdAt : 사진 파일이 등록된 시각
- [Index] : (postUUID, sortOrder), (hash), (groupUUID, createdAt DESC, sortOrder DESC)
- [Unique Key] : (groupUUID, hash)
> 삭제 시, Hard Delete
